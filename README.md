# 📄 Enterprise AI Resume Screening System using NLP

An AI-powered Applicant Tracking System (ATS) built with **Python**, **Streamlit**, and **Natural Language Processing (NLP)** to automate resume screening, candidate ranking, interview scheduling, and hiring analytics.

---

## 🚀 Features

- 🔐 User Authentication (Admin, Recruiter, HR, Candidate)
- 📄 Resume Parsing (PDF, DOCX, TXT)
- 🤖 AI Resume Screening using NLP
- 📊 TF-IDF Resume & Job Description Matching
- 🎯 Candidate Ranking & Scoring
- 🧠 Skill Gap Analysis
- 💼 Job Management Portal
- 👨‍💼 Applicant Self-Service Portal
- 📅 Interview Scheduling
- 📈 Interactive Dashboard & Analytics
- 📋 Candidate Pipeline Tracking
- 📄 PDF Report Generation
- 📊 CSV Report Export
- 🌙 Light & Dark Theme Support

---

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/knauman1488-ai/AI-Resume-Screening-System.git
```

### Navigate to the project

```bash
cd AI-Resume-Screening-System
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run app.py
```
